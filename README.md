# React实战项目－招聘app

> * 服务端框架：NodeJS koa2
> * 数据存储：postgresql
> * 前端框架：react 16.4.2
> * 路由：react-router 4.3.1
> * 状态管理：react-redux
> * 打包工具：webpack 3.8.1

## 1、创建项目
```
npx react-create-app RecruitApp
```
## 2、弹出配置文件，可以自定义配置webpack（可选）
```
yarn run eject
```
## 3、安装依赖
```
cd RecruitApp
yarn add koa2 --save
yarn add redux --save
```
